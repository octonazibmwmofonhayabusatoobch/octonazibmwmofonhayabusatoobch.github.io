Banner Grabbing

Grab it by the banner

Overview

Banner grabbing is a technique used to collect software, service, and version information from network hosts. It helps both security professionals and attackers identify vulnerabilities in systems.

Types of Banner Grabbing

1. Active Banner Grabbing

Active banner grabbing involves connecting to the server and sending requests to retrieve information, which can trigger security alerts.

Telnet

Use Telnet to connect to a server and grab the banner

              telnet example.com  80
                
then type HEAD HTTP/ /1.0 to get the banners

              HEAD/ HTTP/1.0

Nmap

Use Nmap to scan a target for open ports and identify service versions:

              nmap -sV example.com

2. Passive Banner Grabbing

Passive banner grabbing collects information without directly interacting with the target system, making it more stealthy.

cURL Example

Use cURL to silently retrieve HTTP headers from a target:

              curl -s -I example.com

Wappalyzer

Install Wappalyzer as a browser extension for Chrome or Firefox to automatically retrieve web technologies and services used by a website.

Common Ports Targeted

Banner grabbing typically targets the following ports:

  • Port 80 (HTTP)
  • Port 21 (FTP)
  • Port 25 (SMTP)
  • Port 22 (SSH)
  • Port 443 (HTTPS)
  • Port 3306 (MySQL)

Preventing Banner Grabbing

To protect against banner grabbing, organizations should:

  • Hide or modify service banners
  • Disable unnecessary services
  • Implement strict firewall rules
  • Use intrusion detection systems (IDS)
